# Тренажер памяти

Веб-приложение для тренировки памяти с админ-панелью.

## Описание

Тренажер памяти помогает развивать навыки запоминания. Приложение включает:

- **Тренажер** с тремя столбцами:
  1. Картинка
  2. Поле для ввода ответа (число)
  3. Результат проверки

- **Простой уровень**:
  - 30 случайных картинок из диапазона 0-100
  - 2 попытки на каждый вопрос
  - Подсчет верных ответов
  - Отображение результатов в конце

- **Админ-панель**:
  - Просмотр всех доступных изображений
  - Просмотр результатов тренировок
  - Настройки приложения

## Установка

1. Создайте виртуальную среду (если еще не создана):
```bash
python3 -m venv .venv
```

2. Активируйте виртуальную среду:
```bash
source .venv/bin/activate
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

4. Запустите приложение:
```bash
# Вариант 1: Используя скрипт
./run.sh

# Вариант 2: Вручную
python app.py
```

5. Откройте в браузере:
- Тренажер: http://localhost:5000
- Админ-панель: http://localhost:5000/admin

## Использование

1. Выберите уровень сложности на главной странице
2. Вводите число, соответствующее картинке
3. У вас есть 2 попытки на каждый вопрос
4. После завершения тренировки просмотрите результаты
5. Используйте админ-панель для просмотра статистики
